ソフトの概要

　　　・エクセルのファイルを（＊１）
　　　・指定された列のデータ内容（＊２）ごとにシートを分割し（＊３）、
　　　・指定したホルダへ別の名前（＊４）でファイル（＊５）を作成します。

　　　補足
　　　　・＊１：ブックの１シート目
　　　　・＊２：以降、説明上「分割コード」と呼びます。
　　　　・＊３：1行目からタイトル行はそのままコピーされ分割コードの行が以降に入ります。
　　　　・＊４：ファイル名は [分割コード + "_" + もとのファイル名.xlsx]
　　　　・＊５：もとのファイルの2シート目以降はそのままこぴーされます。
　　　その他説明
　　　　・分割対象データはタイトル行の後から分割コードがブランクになるまで
　　　　・分割対象データは分割コードでソートされます。
　　　　・必要な場合、1シート目は編集可開始行とシート保護パスワードを設定できます。

・インストール方法
　　　・参考（内容説明）
　　　　使用方法:
　　　　　　1.ダウンロードしたzipファイルを右クリで「すべて展開(T)」
　　　　　　2.C:\直下等を「参照ボタン」から指定し解凍すると、fgksystemsのホルダができます。
　　　　　　3.このfgksystemsのホルダにあるFGK31ES.exeを実行します。
　　　　　　　(ショートカットを必要に応じて作成ください。)

・アンインストール方法
　　　・レジストリは使っていませんので対応は不要です。
　　　・下記の3つのファイルを削除またはfgksystemsのホルダを削除
　　　　　・FGK31ES.exe
　　　　　・T26Xbase.txt
　　　　　・このreadme_エクセルシート結合.txt

　　　・ショートカットを作成した場合はそれを削除

準備の説明 

・分割したいシートのあるExcelﾌｧｲﾙを入力ホルダに入れます。
　　・ホルダは C:\ 以下の作成してください。
　　・1から9列のいずれかを分割列にする
　　・分割データの最初の行はタイトル行、この下の行がデータ行
　　・分割コードは仕入先コードや客先コードなどをいれる。
　　・分割コードがブランクになったらデータの終わり。
・出力先のホルダを決めるまたは作成。
　　・ホルダは C:\ 以下の作成してください。
・必要な場合、編集可開始行とシート保護パスワードを設定。
 
画面入力の説明 

①参照ボタンから入力ファイルを指定。 
②分割列位置を設定。 
③分割列タイトル名を指定。
④参照ボタンから出力先ホルダを指定。 
⑤編集可開始列とシート保護パスワードを指定。（必要な場合のみ）
⑥実行ﾎﾞﾀﾝを押します。
